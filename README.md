Running the code
----------------

To run a C++ file named ``FILE.c``
  
    g++ FILE.c -o FILE
    ./demo.c
    
To run a C++ file that makes use of the CVC4 libraries, such as ``helloworld.cpp``,

    g++ helloworld.cpp -o helloworld -lcvc4
    ./helloworld
